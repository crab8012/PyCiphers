What is PyCipher?


PyCipher is a multifaceted python program that focuses on encoding and decoding information. It uses a combination of the Python library Flask to serve web pages, handmade Python methods to handle encoding and decoding, HTML to provide a user interface, and CSS to make that user interface look presentable.
